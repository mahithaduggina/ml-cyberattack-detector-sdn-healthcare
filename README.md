# Machine Learning Based Cyberattack Detector in Software Defined Networking for Healthcare Systems

## Project Overview
This project is designed to detect cyberattacks in Software Defined Networking (SDN) environments used in healthcare systems by applying machine learning techniques. The goal is to identify suspicious or malicious network traffic patterns and improve the security of healthcare-related network infrastructure.

This project combines machine learning, cybersecurity, and web-based deployment concepts to create a practical threat detection system.

## Key Features
- Machine learning-based cyberattack detection
- Focused on Software Defined Networking (SDN) environments
- Healthcare system security use case
- Data preprocessing and model-based prediction
- Web application interface for interaction and testing
- Practical integration of ML model with application deployment

## Technologies Used

## Project Structure
- `app.py` → Main Flask application
- `Notebook.ipynb` → Model development and experimentation
- `processed.csv` → Processed dataset
- `model.sav` → Trained machine learning model
- `signup.db` → SQLite database file
- `flowchart.txt` → Project flow / logic notes
- `sample.txt` → Sample input or reference data
- `static/` → Static assets
- `templates/` → HTML templates
- `archive/` → Backup or archived files

## How It Works
1. The dataset is preprocessed and prepared for analysis.
2. A machine learning model is trained to detect suspicious network behavior.
3. The trained model is saved and integrated into the Flask application.
4. The application allows users to interact with the system through a web interface.
5. The system predicts whether the input indicates a potential cyberattack.
